# rectangle.py
- Функция area позволяет вычислить площадь прямоугольника:
    ```python
  import rectangle
  
  print(rectangle.area(2, 5)) # -> 10
    ```
- функция perimeter позволяет вычислить периметр прямоугольника:
    ```python
  import rectangle
  
  print(rectangle.perimeter(2, 5)) # -> 14
    ```
