# triangle.py
- Функция area позволяет вычислить площадь треугольника:
    ```python
  import triangle
  
  print(triangle.area(2, 5)) # -> 5.0
    ```
- функция perimeter позволяет вычислить периметр треугольника:
    ```python
  import triangle
  
  print(triangle.perimeter(2, 5, 7)) # -> 14
    ```
